==========

SUMMARY
=======
- A module that provide message bar on header of the site to attract the user.

REQUIREMENTS
============
- None.

INSTALLATION
============
- To install, copy the assessments directory and all its contents to your modules
  directory. To enable this module, go to Administer > Modules, and
  enable "assessments".

CONFIGURATION
=============
- Go To admin/config/user-interface/assessments
- Add the text message, color and delay time in the settings and you wiill
  see the messagepar or a dropdown on the header of the site.
